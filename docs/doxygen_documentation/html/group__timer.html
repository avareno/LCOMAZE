int timer_display_conf ( uint8_t timer,
uint8_t st,
enum timer_status_field field )

Displays the configuration of a timer.

This function takes a timer number, a status byte, and a field to specify which part of the configuration to display. It extracts the relevant information from the status byte based on the specified field and prints the configuration.

Parameters
timerThe timer number.
stThe status byte containing the timer configuration.
fieldThe field specifying which part of the configuration to display.
Returns
0 on success, 1 if an invalid field is specified.

◆ timer_subscribe_int()

int timer_subscribe_int ( uint8_t * bit_no)

Subscribes and enables interrupts from the timer.

This function subscribes to the timer interrupts by setting the policy for the timer's IRQ line. It also enables the interrupts by setting the corresponding bit in the bit_no variable.

Parameters
bit_noPointer to a variable that will store the position of the hook id bit.
Returns
Returns 0 upon success, 1 if bit_no is NULL.
