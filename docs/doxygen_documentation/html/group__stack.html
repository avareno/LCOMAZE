◆ stack_push()

void stack_push ( struct Stack * stack,
struct Cell * cell )

Pushes a new cell onto the stack.

This function creates a new StackNode, sets its cell field to the provided cell pointer, and its next field to the current top of the stack. It then updates the top of the stack to be the new node.

Parameters
stackA pointer to the stack where the new element should be added.
cellA pointer to the element that should be added to the stack.
