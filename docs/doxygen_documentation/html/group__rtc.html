◆ set_update()

int set_update ( )

Activates the interrupt mode for the Real-Time Clock (RTC).

This function sets the interrupt enable bit in the RTC's Register B, allowing the RTC to generate interrupts. The interrupt is triggered when the RTC's internal clock reaches the value set in the alarm registers.

Returns
0 on success, non-zero otherwise.

int subscribe_rtc ( uint16_t * bit_no)

Subscribes to RTC interrupts.

This function subscribes to RTC interrupts by setting the policy for the RTC IRQ line. It also retrieves the bit number associated with the RTC hook ID.

Parameters

◆ unsubscribe_rtc()

int unsubscribe_rtc ( )

Unsubscribes the RTC interrupts.

This function unsubscribes the RTC interrupts by calling sys_irqrmpolicy() with the RTC's hook_id. If an error occurs during the unsubscription process, an error message is printed and the function returns 1.

Returns
0 upon success, 1 upon failure.
