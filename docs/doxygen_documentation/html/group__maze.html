◆ generate_maze_buffer()

void generate_maze_buffer ( struct Maze * maze)

Generates a buffer for the maze.

This function checks if the maze buffer is NULL and allocates memory for it if necessary. The buffer is then used to optimize the drawing of the maze by drawing the entire maze to the buffer first and then copying only the visible portion to the screen.

Parameters
mazeA pointer to the Maze structure.
